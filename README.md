# Ellie's Game

## Game Modes:

1. ABCs
    - Runs through the English Alphabet currently only Capital letters
2. Name Spelling
    - Spells Ellie's first and last name
3. 123s
    - counts from 0 to 9
4. Counting
    - counts from 0 to 1000

## Future Modes:

- DinoMatch
- DinoSpelling
- Quantity Match
