# Nora's Game

## Game Modes:

1. ABCs
    - Runs through the English Alphabet currently only Capital letters
2. Name Spelling
    - Spells Nora's first and last name
3. 123s
    - counts from 0 to 9
4. Counting
    - counts from 0 to 1000
5. DinoMatch
    - fun memory game with dino cards
6. DinoRacer
    - connects to google's dinoracer game

## Future Modes:

- DinoSpelling
- Quantity Match
